## Get Started

Put videos in ***data_samples/original*** folder and modify the data index tree as following:

```
├── [data_samples]
│   ├── original
│   │   ├── videoname1
│   │   │   ├── info.txt
│   │   │   ├── framename11.png
│   │   │   ├── framename12.png
│   │   ├── videoname2
│   │   │   ├── info.txt
│   │   │   ├── framename21.png
│   │   │   ├── framename22.png
```

The video info file ***info.txt*** records the path and timestamp ($\mu s$) of each frame.
